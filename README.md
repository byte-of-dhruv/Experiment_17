# Experiment_17
AIM

To study and implement basic data visualization techniques in Python using Matplotlib and Seaborn for representing trends, comparisons, distributions, and relationships in data.

THEORY

Data visualization is the graphical representation of data to understand patterns, trends, and relationships effectively. Python libraries such as Matplotlib and Seaborn provide powerful tools for creating various types of charts like line graphs, bar charts, histograms, and scatter plots. These visualizations help in analyzing data trends over time, comparing values across categories, understanding data distribution, and identifying relationships between variables. Seaborn enhances visualization by providing better aesthetics and simpler syntax for statistical plots.

ONE-LINE EXPLANATIONS OF FUNCTIONS

plot() – Creates a line chart to show trends in data.

figure() – Sets the size and layout of the plot.

xlabel() – Sets the label for the x-axis.

ylabel() – Sets the label for the y-axis.

title() – Adds a title to the graph.

legend() – Displays labels for multiple data series.

bar() – Creates a bar chart for comparison of values.

text() – Adds value labels on bars.

grid() – Displays grid lines on the chart.

xticks() – Sets custom labels for x-axis values.

hist() – Creates a histogram to show data distribution.

axvline() – Draws a vertical line (e.g., mean line) on a plot.

scatter() – Creates a scatter plot to show relationships between variables.

show() – Displays the generated plot.

lineplot() – Creates a line graph using Seaborn.

barplot() – Creates a bar chart using Seaborn.

histplot() – Creates a histogram using Seaborn.

scatterplot() – Creates a scatter plot using Seaborn.
